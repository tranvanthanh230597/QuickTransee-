# Usage

This application provides the following core translation features:

---

## Translation by Selection
Select text and press the shortcut to translate

---

## Translate by Input
 Press shortcut to open translation window, translate by hitting Enter

---

## Translation from Clipboard
More efficient workflow by integrating other apps.

---

## Screenshot OCR
Press shortcut, select area to OCR

---

## Screenshot Translation
Press shortcut, select area to translate

---
